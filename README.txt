Die Version der Software 'SolarSystemSimulation' enthält noch
keine Bedienungsanleitung. Diese wird erst im weiteren Verlauf
des SE-Projektes ausgearbeitet und in das Programm integriert.

In weiten Teilen erklärt sich das Produkt von selber. Lediglich
zur Kameraführung sind spezielle Tastenkombinationen nötig:

STRG + WINDOWS-TASTE + Mausbewegung : Die 3d-Szene wird um den
Ursprung gedreht. Drehen des Mausrades ändert die Entfernung
zm Ursprung.

Alternativ kann statt der Windows-Taste auch die ALT-Taste
verwendet verden. Bei VMWare ist diese Tastenkombination aber
mit dem verlassen der virtuellen Umgebung verbunden.

STRD + SHIFT + Mausbewegung : Die 3d-Szene wird verschoben.
Drehen des Mausrades ändert ebenfalls die Entfernung zum Ursprung.

Anstelle der Mausbewegung können auch die Pfeiltasten verwendet
werden. Bild nach oben und Bild nach unten entsprechen der
Entfernungsänderung.


Das Passwort für den user 'user' ist :   se-project

Das root-Passwort lautet ebenfalls :     se-project


Der PostgreSQL-Datenbank-Server startet automatisch beim booten.

Als Administrationswerkzeug für die PostgreSQL-Datenbank ist
pgadmin3 installiert. Dies kann bei Bedarf mit dem blauen
Elefanten-Icon in der Task-Leiste gestartet werden.
