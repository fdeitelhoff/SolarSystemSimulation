Detailed Description

A class for encapsulating all data for a gluPerspective and a gluLookAt call. These include: _Fovy: Opening angle of frustum in y-direction (45Â° is a good average to start with) _Aspect: The aspect ratio of the viewport (width / height) _Near and _Far clipping plane distances _Center: The 3d-point to appear in the center of the viewport _Camera: The 3d-point where the camera is _Up: The 3d-vector that points upwards in the viewport

Use the apply() function to transfer the perspective settings to the OpenGL matrices.
